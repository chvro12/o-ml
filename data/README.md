# Dataset Jigsaw

`comments.csv` est le fichier d'entraînement Jigsaw Toxic Comment
Classification, téléchargé depuis le dépôt Hugging Face public :

```text
https://huggingface.co/datasets/thesofakillers/jigsaw-toxic-comment-classification-challenge/resolve/main/train.csv
```

Il contient la colonne `comment_text` utilisée par le script de charge et six
colonnes de toxicité utilisées pour entraîner les modèles. Le sujet indique
une licence CC0 pour ce dataset.
